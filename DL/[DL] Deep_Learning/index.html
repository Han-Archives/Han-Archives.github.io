Deep Learning

딥러닝은 신경망 기반의 복잡도가 큰 머신러닝 모델.

많은 수의 은닉층을 사용하여 마치 인간 뇌의 뉴런과 비슷한 방식의 인공신경망 방식으로 정보를 처리

인공신경망

인공신경망이란 생물학의 신경망에서 영감을 얻은 통계학적 학습 알고리즘

[출처] : http://scienceon.hani.co.kr/397536

신경망 아키텍처

인공 신경망은 아래와 같은 신경망 구조로 발전되어 다중 신경망을 구성한다.

[출처] https://www.asimovinstitute.org/neural-network-zoo/

그 외에도 dropout 방식과 같은 다양한 구조가 존재한다.

딥러닝을 이해하기 위한 사전 지식으로 활성화 함수,손실함수, 옵티마이저에 대해 알아보자.

활성화 함수

활성화 함수는 신경망 구조에 비선형성을 추가하기 위해 존재한다.

선형구조의 경우 많은 계층을 거치는 과정에서 전체 신경망 모델이 단순해진다.

이를 방지하기 위해 비선형 구조로 변환하는 것이다. 다만 비선형 구조의 경우에도 기울기 소실과 같은 문제점이 존재한다.

활성화 함수의 종류

시그모이드 함수

TanH 함수(하이퍼볼릭탄젠트 함수)

ReLU

Softmax

손실 함수

손실 함수란 예측한 값이 실제값과의 차이를 수치화하는 함수를 말한다. 이 손실함수를 최소화하는 가중치와 편향을 찾는 것이 중요 목표이다.

손실 함수 종류

  • Binary Cross-Entropy

Sigmoid 함수를 사용하는 이진 분류의 경우 주로 사용

  • categorical_crossentropy (범주형 교차 엔트로피)

Softmax 함수를 사용하는 다중 클래스 분류(Multi-Class Classification)의 경우에 주로 사용

  • sparse_categorical_crossentropy

위의 범주형 교차 엔트로피와 동일하게 다중 클래스 분류에 사용되지만 원-핫 인코딩된 상태가 아닌 정수형값을 (0,1,2,3,4) 갖는 레이블에 대해 수행

  • MSE

평균제곱오차라 하며 오차(실제값과 예측값의 차이)를 제곱한 값을 나타내는 손실함수

옵티마이저

옵티마이저란 아래에서 언급될 역전파 과정에서 가중치를 수정할 때 사용하는 알고리즘으로

옵티마이저 종류

  • GD(경사 하강법)

전체 데이터를 계산하여 발생한 오류를 통해 가중치를 수정. -> 오래걸리고 메모리 소모가 큼

  • SGD

배치 단위로 가중치를 수정

  • Momentum

관성이라는 물리학 법칙을 응용한 것으로 경사하강법의 기울기 + 그 전 시점의 기울기의 일정비율을 더하는 것

모멘텀을 통해 아래와 같이 기울기가 0인 구간(로컬 미니멈)을 돌파하여 우리가 원하는 가장 낮은 손실함수 값에 도달할 수 있음

  • Adagrad

각 매개변수에 서로 다른 학습률 적용하는 것으로 변화가 많은 매개변수는 학습률을 작게, 변화가 적은 매개변수는 학습률을 크게 설정

  • RMSprop

Adagrad의 단점인 학습이 계속 진행되면 나중에는 학습률이 크게 떨어진다는 단점을 다른 수식으로 개선한 것

  • Adam

RMSprop + Momentum

가장 많이 사용되는 옵티마이저로, Momentum(방향)과 RMSprop(학습률)을 조정

다중 신경망

인공 신경망의 조합을 통해 다중 신경망을 생성하며 이 신경망들은 아래의 과정을 통해 다음 노드로 전달 되거나 결과값을 나타낸다.

입력 값들은 설정된 가중치와 곱해지고 bias를 더한 뒤 활성화 함수를 통해 값이 변화된 후 다음 layer로 전달된다.

역전파

역전파란 딥러닝의 핵심 요소 중 하나로 기존에 진행한 방식에서 발생한 오차를 토대로 손실함수를 최소화하기 위해 가중치를 수정하며 진행하는 방식

예시

아래의 예시는 예능에서 자주 하는 맞추기 게임으로 각 사람은 정답에 대한 설명을 통해 전달하여 최종 사람이 답을 맞추는 과정이다. 이 때의 설명을 가중치로 이해하면 역전파의 과정을 이해하기 더 쉬울 것이다. 역전파 과정이 끝이 나면 다시 수정된 내용을 기반으로 다시 진행하는 과정을 통해 손실함수를 줄여가며 개선하며 진행한다.
